JavaScript sem build tools

Foi implementado uma aplicação simples de relógio usando diferentes frameworks para ser possível comparar a complexidade de implementação em cada um e o tamanho do download que o browser deve fazer para executar cada um.

Mais informações no README no repositório do projeto: guilhermerodrigues680/js-without-build-tools

Implementações

  • Alpine.js
  • Lit (ESM)
  • Mithril.js
  • Petite-Vue
  • Preact
  • React
  • SolidJS (ESM)
  • Vue.js 3

É possível executar todas as implementações apenas colocando o path completo do arquivo no browser.
As aplicações que usam ECMAScript Modules (ESM) podem necessitar de um servidor web como nginx, python http.server, etc, se forem usados modulos locais: JavaScript modules | MDN
